# Employee Performance and Satisfaction Analysis

## Objective

Analyze employee performance, satisfaction, and workplace trends.

## Dataset

Employee Dataset (10000 records)

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* Seaborn
* Matplotlib
* Scipy

## Project Phases

1. Data Understanding
2. Statistical Analysis
3. Trend Investigation
4. Correlation Analysis
5. Hypothesis Testing
6. Employee Segmentation
7. Business Insights

## Key Findings

* Employee salary, performance, and satisfaction levels were analyzed across different departments.
* Trend analysis helped identify relationships between experience, salary growth, and employee satisfaction.
* Correlation analysis showed patterns among training hours, work hours, performance, and salary.
* Hypothesis testing using T-Test was performed to evaluate whether overtime impacts employee satisfaction.

## Conclusion

This project explored employee performance and satisfaction using statistical analysis and visualization techniques. The analysis provided insights into employee behavior, workplace trends, and performance factors. Based on the hypothesis testing results, overtime hours did not show a statistically significant impact on employee satisfaction in the selected dataset. Employee segmentation and trend analysis can support better decision-making and improve workforce management strategies.
